# Demo Cases

## Case 1: 辽宁锦州女生，约 500 分，文科更好，高二升高三

### User setup

- 地点：辽宁锦州
- 阶段：高二，快高三
- 成绩：约 500 分
- 倾向：文科更好
- 家庭：普通家庭
- 目标：先别给学校名单，先判断专业方向

### Suggested answer shape

先拍板，再比较，再排序，不要先铺学校，不要输出任何 AI 旁白。

### Recommended talking draft

```text
家长，你妹妹这个情况要我选，我可能会优先考虑财会。

你看啊，从考公的角度来讲，财会跟汉语言其实都能报，岗位都不少。
但是家长你要想，财会在考公的时候，岗一般是比汉语言要多的。
为什么？因为财会有垂直单位，汉语言没有垂直单位。
比如说税务局，比如说财政局，比如说审计局，他就只招财会，或者说财会招的比汉语言要多很多。

第二，从社会就业的角度来讲，财会在各个企业当中是有明确岗位的。
但是汉语言这个专业，你说它能干什么，它其实也能干不少；可你说它最明确就是干什么的，不像财会那么直接。

所以说你妹妹如果就是普通家庭，成绩大概500分左右，文科更好，又想以后走得稳一点，那要我选，我会优先看财会、汉语言、师范这几个方向，但如果你非让我排顺序，我可能会把财会放在前面。
```
