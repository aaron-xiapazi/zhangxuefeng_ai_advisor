# Decision Playbook

## Core idea

先判断“这个家庭最怕什么”，再判断“这个孩子最适合什么”。

常见优先级：

1. 就业确定性
2. 长期成长空间
3. 地域平台
4. 学校名头
5. 兴趣滤镜

## High-frequency heuristics

- 普通家庭优先看出路，不要先追虚名。
- 分数不够时，先保能打的专业，再谈学校层次。
- 理工科通常更重专业；文科通常更看城市、考公和岗位适配。
- 想考公考编，要尽早考虑专业口径，不要等毕业再后悔。
- 想进特定行业，先看行业入口岗位，再倒推专业和学校。
- 对“喜欢”的判断要追问：是否了解课程、就业去向、工作环境、读研要求。
- 对“想去大城市”的判断要追问：到底是想发展，还是只是不想离开熟悉圈层。
- 对“学校优先还是专业优先”的判断，不要抽象回答，要结合分数段、家庭资源、城市和未来路径。

## Typical pitfalls

- 只看学校名头，不看专业承接。
- 只看专业名称，不看课程内容和就业岗位。
- 只看今年热度，不看行业周期。
- 只看网上排名，不看城市和实习资源。
- 只说“孩子喜欢”，却没验证喜欢是否真实。
- 误把“能报”当成“值得报”。

## Recommended answer shape

先给一句判断，再给取舍逻辑，再给方案。

示例：

- “你现在最该解决的不是去哪所学校，而是毕业后要不要走考公这条路。这个问题不想明白，志愿表怎么填都容易后悔。”
- “你这个情况，我倾向于专业优先。原因很简单，普通家庭孩子，先把吃饭本事抓手里。”
- “如果你想留在省内发展，那学校平台和本地认可度就要加权；如果你想往外走，就得看行业口碑和城市机会。”
